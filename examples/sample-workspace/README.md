# KanbanRPM Sample Workspace

This sample workspace is a general research-operations demo for first-time KanbanRPM users.

## How To Use

1. Install KanbanRPM in an Obsidian vault.
2. Copy the `KanbanRPM Workspace/` folder from this directory into the root of that vault.
3. Run `KanbanRPM: Open board`.
4. Explore `Board`, `Timeline`, `Gantt`, `Archive`, communication notes, routines, and small actions.

## What This Demonstrates

- `Project -> Subproject -> Big Action -> Small Actions`
- `Current Focus`, `Waiting`, `Blockers`, `Flow`, and `Timeline`
- Status lanes: `Inbox`, `Active`, `Waiting`, `Blocked`, `Someday`, `Done`
- Scheduled dates, due dates, next reviews, and routines
- Communication source notes and yearly communication logs
- Experiment/Analysis style research logs without requiring a specific discipline

The examples are fictional and intentionally general.
